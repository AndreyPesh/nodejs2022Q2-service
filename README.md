# Home Library Service

## Downloading

```
git clone {repository URL}
```

## Installing NPM modules

```
cd nodejs2022Q2-service
git checkout auth

```

### build images and run multi-container
```
  npm run docker
```

### host
```
  http://localhost:4000/
```

### run test
```
  npm run test:auth
```

### run lint
```
  npm run lint
```

### swagger UI
  [localhost:4000/doc](http://localhost:4000/doc)
